# Тестовое задание на позицию дата инженера

## Что реализовано
**Данный репозиторий полностью закрывает основные поставленные задачи, а именно:**
- Проектирование двух слоёв: STG и DDS;
- Разработку потоков перекладки и трансформации данных: ELT 1 и ELT 2.

**Помимо этого, дополнительно создан небольшой даг-файл (`SCRIPTS/EXTRA/vault_dag.py`) для оркестрации пайплайна при помощи Airflow.**
*Реализованный файл не предназначен для выполнения на Win-машине. Для полноценной работы необходимо законнектить Airflow и PostgreSQL*

---

## Быстрая навигация по репо
- В `SCRIPTS/ETL/...` расположены ETL файлы;
- В `SCRIPTS/EXTRA/...` находится extra-решение (файл оркестрации);
- В `SQL/...` расположены DDL скрипты;
- В `SQL/CHECKS.md` оставил скрипты для быстрой проверки рабьоты.

---

## Перед работой:

### 1. Создайте виртуальное окруженик

```bash
python -m venv venv
```

### 2. Активируйте виртуальное окружение

**WIN:**
```PowerShell
.\venv\Scripts\Activate.ps1
```

**macOS/Linux:**
```bash
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Создайте и заполните `.env` по примеру `.env/example`
